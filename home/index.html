POST /todoapi

Creates a new todo.

Parameters in body

  • title (string): The title of the todo.
  • description (string): The description of the todo.
  • stautus (string): The stautus of the todo.

Response

                "item":     {
                    "title": " Todo 4",
                    "description": "Example Todo 4",
                    stautus:"pending"
                }
