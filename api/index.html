Todo API Documentation

GET /todos

Retrieves a list of all todos.

Parameters

  • None

Response

{
    "todos": [
        {
            "id": 1,
            "title": "Example Todo 1",
            "completed": false
        },
        {
            "id": 2,
            "title": "Example Todo 2",
            "completed": true
        }
    ]
}

POST /todos

Creates a new todo.

Parameters

  • title (string): The title of the todo.

Response

{
    "id": 3,
    "title": "New Todo",
    "completed": false
}
